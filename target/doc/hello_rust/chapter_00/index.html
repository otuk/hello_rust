This chapter is only about one thing; getting code running in rust.

This is the file structure for our project so far:

hello_rust 
├── Cargo.lock
├── Cargo.toml
└── src
    ├── chapter_00
    │   └── mod.rs
    │
    └── main.rs

Cargo.lock is not to be hand edited, it reflects Cargo.toml file contents at a certain point in time. More on this later. Cargo.toml file is where we describe the hello_rust project and its dependencies, for now it is just a boiler plate file. The src directory is where our source code is. We have edited the main.rs source file and the chapter_00/mod.rsfile we are in. The directory chapter_00 is our chapter 0 module directory.

Rust allows us to break our code in separate functions, files, modules and crates. We are in hello_rust crate, chapter_00 module, and mod.rs file to execute the print_the_ubiquitous_hw() function.

To run this project all we need to do is execute cargo run from the project root hello_rust directory.
