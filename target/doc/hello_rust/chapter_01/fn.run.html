These function will execute sample code relevant to each chapter.

Remarks

This function will introduce some variables. Using let x = 2 is a simple way to create a variable on stack. Rust infers the local variables types, but you can also specify.

All variables are immutable by default. To have a variable that you can change you have to use mut keyword.
